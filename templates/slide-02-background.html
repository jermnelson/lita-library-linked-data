Background

Expected Outcomes, Tools, Schemas, and the Semantic Web

Expected Outcomes for this Webinar

In this webinar we will be using the Python and Javascript programming languages to manipulate MARC21 and MARCXML into BIBFRAME, Dublin Core, and Schema.org graphs made-up of RDF triples in order to expose this linked data in the form of RDF xml, RDF N-triples, HTML5 RDFa, HTML5 microdata, and JSON Linked Data.

We will be using the
following Tools

Python modules:

  • Flask
  • Jinja2 (part of Flask)
  • lxml
  • PyMARC
  • RDFLib

Javascript libraries and frameworks:

  • Bootstrap
  • jQuery
  • Knockout.js

Schemas, Formats, and Structured Data

Schemas

  • Schema.org with extension efforts by Schema Bib Extend Community Group
  • Dublin Core Metadata Initiative
  • BIBFRAME
  • MODS (Metadata Object Description Schema)

Formats

  • MARC Standards
    • Bibliographic data
    • Authority data

Structured Data

  • RDF (Resource Description Framework)
    • RDF XML
    • RDF N-triples
    • RDF Turtle
  • RDFa
  • Microdata
  • JSON Linked Data
