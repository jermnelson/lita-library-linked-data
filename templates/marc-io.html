Extracting Data from MARC21 and MARCXML

MARC21

MARC21 is a binary format developed by the Library of Congress. Dealing with the raw MARC21 can be challenging because MARC21 combines both fixed-length and variable length fields. Fortunately a number of open-source libraries exist for manipulating MARC21 records that hides some of the complexity behind the format. Today we will use Ed Summer's PyMARC module for coding experiment.

Task #1 -- Reading MARC21 file

    $ >> import pymarc
    $ >> marc_reader = pymarc.MARCReader(open('E:\\training\\marc-sample.mrc'))
    $ >> marc_records = []
    $ >> for record in marc_reader:
    $ >>     marc_records.append(record)
