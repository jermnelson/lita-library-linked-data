Downloads

All source code and the webinar's IPython Notebook is available on github at https://github.com/jermnelson/lita-library-linked-data. To download all of the MARC record samples and the IPython Notebook, you can download and extract a zip file of the repository here.

IPython Notebook

To earn your Open Badge follow the directions on the IPython installation page to get a working IPython environment. I also recommend you download/install a version of the pip, the Python package installer by downloading get-pip.py and then running the following from your command-line where you downloaded the get-pip.py file: 

python get-pip.py

After installing pip, you should be able to install the remaining modules by entering the following commands:

NOTE for Microsoft Windows! First install lxml from this list of windows installers. You'll receive an error if you try to install lxml with pip 

    pip install jinja
    pip install pymarc
    pip install rdflib

Macintosh & Linux ONLY!

    pip install lxml

MARC21 and MARCXML Samples

The coding experiments illustrated in this webinar use both MARC21 and MARCXML records. Download and unzip the following files (right-mouse click and "Save Link as.." to a directory on your computer)

  • marc-sample.mrc Collection of close to 10,000 MARC21 records from Colorado College's collection.
  • marc-sample.xml Collection of MARC XML records derived from marc-sample.mrc.
