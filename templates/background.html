  • Schema.org with extension efforts by Schema Bib Extend Community Group
  • Dublin Core Metadata Initiative
  • BIBFRAME
  • MODS (Metadata Object Description Schema

Formats

  • MARC Standards
    • Bibliographic data
    • Authority data

Structured Data

  • RDFa
